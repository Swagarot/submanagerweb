# Expense Manager — Streamlit

A web-based expense manager built with Streamlit.

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## What it does

- Add, edit, delete, and duplicate expenses
- Track expenses by category
- Support one-time, monthly, and yearly expense types
- Convert currencies using live exchange rates
- Search, filter, and sort expenses
- Save and load expense data as JSON
- Load sample expenses for testing

## Main files

- `streamlit_app.py` — the Streamlit web application
- `dummy_manager.py` — sample expense data loader used by the app

## Notes

The current repository is focused on the Streamlit web version. Legacy CLI module descriptions are no longer relevant for this version.
